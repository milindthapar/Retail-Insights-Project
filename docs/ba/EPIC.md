# EPIC: End-to-End Retail Revenue Insights Platform  
**Project Scope:** Analysis, forecasting, customer insights, cloud-based reporting & automated refresh  
**Owner:** Business Analyst – Milind Thapar  
**Stakeholders:** Retail Operations, Finance, Sales Leadership, BI Team

---

## 1. Epic Summary
Build an end-to-end analytics platform that helps retail decision-makers understand revenue, profit, discounts, customer value, and product performance—powered by a cloud data warehouse (Azure SQL), automated Python ETL, and a dynamic Power BI dashboard.

---

## 2. Business Goals
- Provide a **single source of truth** for all sales data  
- Enable **daily automated refresh** without manual Excel work  
- Deliver **profitability insights** across customers, products, and regions  
- Identify **discount leakage** and low-margin categories  
- Forecast future performance to support **inventory and budgeting decisions**  
- Provide business leadership with a **real-time dashboard**

---

## 3. Problem Statement
Retail data was stored in multiple local files and required manual cleaning.  
Decision-makers lacked:
- Consolidated visibility of sales  
- Trend analysis  
- Customer segmentation  
- Product profitability insights  
- Forecasts for planning  
- Automated data refresh

This epic addresses all of these by building a modern analytics solution.

---

## 4. Epic Completion Criteria (Definition of Done)
- Azure SQL database created and populated  
- Python ETL developed + automated via Task Scheduler  
- 5-page Power BI report built with validated metrics  
- Python ARIMA forecasting implemented  
- Documentation delivered (BA + technical)  
- GitHub repository published with full structure  
- Stakeholders can view refreshed metrics daily
